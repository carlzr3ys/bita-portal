Cloud Computing Fundamentals

Cloud computing is the delivery of computing services including servers, storage, databases, networking, software, analytics, and intelligence over the Internet ("the cloud") to offer faster innovation, flexible resources, and economies of scale.

Key Concepts

  • On-Demand Self-Service: Users can provision computing capabilities automatically without human interaction.
  • Broad Network Access: Services are available over the network via standard mechanisms.
  • Resource Pooling: Provider's computing resources serve multiple consumers using a multi-tenant model.
  • Rapid Elasticity: Capabilities can be elastically provisioned and released to scale with demand.
  • Measured Service: Cloud systems automatically control and optimize resource use through metering capabilities.

Service Models

IaaS (Infrastructure as a Service) ├── Virtual machines ├── Storage └── Networking PaaS (Platform as a Service) ├── Development tools ├── Database management └── Application hosting SaaS (Software as a Service) ├── Email services ├── CRM systems └── Web applications

Cloud Services

Major cloud providers offer a wide range of services to meet different business needs.

AWS Services

  • EC2: Elastic Compute Cloud for virtual servers
  • S3: Simple Storage Service for object storage
  • RDS: Relational Database Service
  • Lambda: Serverless compute service

Azure Services

  • Virtual Machines: Scalable compute resources
  • Blob Storage: Object storage solution
  • SQL Database: Managed database service
  • Functions: Serverless compute

Example: Creating an S3 Bucket

# Using AWS CLI aws s3 mb s3://my-bita-bucket # Upload a file aws s3 cp file.txt s3://my-bita-bucket/ # List bucket contents aws s3 ls s3://my-bita-bucket/

Linux Basics

Linux is an open-source operating system widely used in cloud computing environments.

Essential Commands

# Navigation cd /path/to/directory # Change directory pwd # Print working directory ls -la # List files with details # File Operations mkdir directory_name # Create directory touch file.txt # Create file cp source dest # Copy file mv old new # Move/rename file rm file.txt # Remove file # System Information whoami # Current user uname -a # System information df -h # Disk usage free -h # Memory usage

File Permissions

Linux uses a permission system with three types: read (r), write (w), and execute (x).

# Change permissions chmod 755 script.sh # rwxr-xr-x chmod +x script.sh # Add execute permission # Change ownership chown user:group file.txt

Networking

Understanding networking is crucial for cloud computing and DevOps.

Basic Network Concepts

  • IP Address: Unique identifier for devices on a network
  • Subnet: Logical subdivision of an IP network
  • Port: Communication endpoint
  • DNS: Domain Name System translates domain names to IP addresses

Common Network Commands

# Check network configuration ifconfig # Network interface configuration ip addr # Modern alternative to ifconfig # Test connectivity ping google.com # Test connection curl https://api.example.com # HTTP request # Network troubleshooting netstat -tulpn # Network connections traceroute google.com # Trace route to destination

Firewall Rules

# UFW (Ubuntu Firewall) sudo ufw allow 22/tcp # Allow SSH sudo ufw allow 80/tcp # Allow HTTP sudo ufw enable # Enable firewall # Check status sudo ufw status

DevOps

DevOps combines software development (Dev) and IT operations (Ops) to shorten the development lifecycle.

Key DevOps Tools

  • Git: Version control system
  • Docker: Containerization platform
  • Kubernetes: Container orchestration
  • CI/CD: Continuous Integration/Continuous Deployment

Docker Basics

# Build an image docker build -t my-app . # Run a container docker run -d -p 8080:80 my-app # List containers docker ps # View logs docker logs container_id # Stop container docker stop container_id

Git Workflow

# Initialize repository git init # Add files git add . # Commit changes git commit -m "Initial commit" # Push to remote git push origin main # Create branch git checkout -b feature-branch
